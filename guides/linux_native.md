# Linux using Docker (Native)

- Ensure the following are installed through your distro's package manager:
  - Docker CE
  - docker-compose
  - git
  - python 3
  - pip
- Install the following
  ```bash
  python3 -m pip install --user pipx
  pipx install copier
  pipx install invoke
  pipx install pre-commit
  pipx ensurepath
  ```
